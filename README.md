Project Title
C# Console Chatbot System
Description
This project is a console based chatbot application developed using C#. The program interacts with users through a text based interface, providing responses and guiding the user through a simple menu system.
The chatbot includes features such as greeting the user, handling menu selections, and performing basic tasks based on user input. The goal of this project is to demonstrate understanding of C# fundamentals, including classes, methods, input/output handling, and program structure.
Features
Voice greeting (WAV file support)
Text-based chatbot interaction
Menu system with options (1, 2, 3)
User input handling and validation
Structured using classes and methods

Technologies Used
C# (.NET)
Console Application
Visual Studio / VS Code
